All minor versions are backward compatible, meaning that you can upgrade to a newer minor version without breaking your code.

However, you should always check the [changelog](/docs/changelog) for any breaking changes.

## Java Version

Fluxzero requires **JDK 21 or higher** to compile and run. It is actively tested on **JDK 24** and remains
compatible with recent versions.

## Versioning Policy

Fluxzero aims to stay up-to-date with its core dependencies. We strive to:

- Use the **latest stable versions** where possible,
- Avoid breaking backward compatibility for common transitive consumers,
- Update frequently, especially for frameworks like **Jackson**, **Spring**, **Hibernate Validator**, and **JUnit**.
