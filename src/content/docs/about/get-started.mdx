## Scaffold a new project

Prerequisites:
- Java 21+

<Steps>

1. Install the Flux CLI

   ```shell
      curl -o- https://github.com/flux-capacitor-io/flux-cli/releases/latest/download/install.sh | bash
   ```

2. Generate a new project

   ```shell
      ~/.flux/cli new {your project name}
   ```

</Steps>
